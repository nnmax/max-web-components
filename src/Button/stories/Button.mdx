# Button

按钮是提供给用户触发某种事件或某种操作的最小组件。比如，打开对话框、执行取消操作、执行删除操作和提交表单等。

## Disabled

Disabled 状态代表当前操作对用户不可用。该状态的按钮会变成灰暗色，并被排除在焦点顺序之外（例如无法点击和使用 <kbd>Tab</kbd> 键获取焦点）。

## ARIA

(WAI-ARIA: https://www.w3.org/WAI/ARIA/apg/patterns/button/)

- 请始终设置 `aria-label` 或 `aria-labelledby` 属性。

  - `aria-label` 或 `aria-labelledby` 可以让屏幕阅读器更好地理解该按钮的作用，尤其是一些图标按钮。

- 键盘交互中的焦点控制。

  - 使用键盘操作按钮并打开一个对话框，则键盘焦点应该被转移到对话框内。

  - 对话框关闭后，键盘焦点应该回到打开对话框的按钮上。
